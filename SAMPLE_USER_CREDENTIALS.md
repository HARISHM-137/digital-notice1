# Sample User Credentials

## 🔐 How to Create Users in Supabase Auth

These users need to be created in **Supabase Authentication** first, then the SQL data will link to them.

### Method 1: Via Supabase Dashboard (Recommended)
1. Go to **Authentication** → **Users** in Supabase Dashboard
2. Click **Add User**
3. Enter email and password from the list below
4. Click **Create User**
5. Copy the **User ID** that's generated
6. Update the `SAMPLE_DATA.sql` file with this User ID

### Method 2: Via API (Bulk Creation)
Use the sign-up endpoint for each user, or use Supabase Admin SDK.

## 📋 Setup Instructions

### Step 1: Create Schema
Upload and run `CLEAN_SLATE_SCHEMA.sql` in Supabase SQL Editor

### Step 2: Create Auth Users
For **each user above**, you need to:

**Option A: Manual (Supabase Dashboard)**
1. Go to Authentication → Users
2. Click "Add User"
3. Enter the email and password
4. Save the generated UUID

**Option B: Via Sign Up Flow**
Just use your app's sign-up page with these credentials

### Step 3: Update User IDs
If the UUIDs in Supabase Auth don't match the ones in `SAMPLE_DATA.sql`, update them.

The easiest way is to let Supabase Auth generate the UUIDs, then run this query to see them:
```sql
SELECT id, email FROM auth.users;
```

Then update the INSERT statements in `SAMPLE_DATA.sql` with the correct UUIDs.

### Step 4: Run Sample Data SQL
Upload and run `SAMPLE_DATA.sql` in Supabase SQL Editor
